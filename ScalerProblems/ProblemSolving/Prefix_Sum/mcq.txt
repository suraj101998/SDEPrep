2. What is the time complexity of creating the prefix sum array of an array A of N integers ? O(N)
3. Given the prefix sum of an array, prefSum = [-2, 4, 1, 5, 2] What is the sum of the original array from index 0 to 2 (0-based) ? 1
4. Given the prefix sum of an array, prefSum = [-2, 4, 1, 5, 2] What is the sum of the original array from index 2 to 4 (0-based) ? -2
5. Given the prefix sum of an array, prefSum = [-2, 4, 1, 5, 2] What is the sum of the original array from index 1 to 2 (0-based) ? 3
